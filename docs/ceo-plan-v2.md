# CEO Plan: Cortex — Personal Workspace

Generated by /plan-ceo-review on 2026-06-03 (Round 2)
Mode: SCOPE EXPANSION → Workspace MVP

## Vision

Cortex — self-hosted personal workspace. Единое пространство где курсы, заметки, задачи и календарь управляются через UI и через Hermes CLI/API. TS-стек, Hermes-first архитектура (агент как оркестратор), local-first, open-source.

**10x Check:** Не просто ещё один AI workspace (как Odysseus) — а workspace где AI-агент (Hermes) является полноценным управляющим слоем: создаёт задачи, пишет заметки, планирует календарь, отслеживает прогресс обучения. Человек и агент работают в одном пространстве.

## Workspace MVP Scope
- **Courses:** Done (Phase 1-5)
- **Notes:** Markdown-редактор с AI, Obsidian vault sync, Hermes R/W
- **Tasks:** Аггрегация из Redmine/Reminders/Telegram, CRUD, статусы
- **Calendar:** События, привязка к задачам/курсам, daily/weekly view
- **Hermes Orchestrator:** Единый API — `cortex ask "что у меня сегодня?"` → сводка из всех модулей
