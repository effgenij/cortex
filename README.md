# Cortex

Персональное рабочее пространство для совместной работы с AI-агентом Hermes.

> **Статус:** проектирование (planning). Код не написан.

## Документация

- [Дизайн-док v4](docs/office-hours-design-v4.md) — текущий план (React + Vite + Mantine + Hono + SQLite)
- [CEO Plan v2](docs/ceo-plan-v2.md) — предыдущая итерация (archive)

## Идея

Workspace где человек и Hermes работают в одном пространстве, на одном языке, с одними данными. Чаты, страницы, блоки — как Notion, но с AI-агентом который сам наполняет контент.
